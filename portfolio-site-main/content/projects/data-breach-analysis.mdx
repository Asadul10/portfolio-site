# Project Overview

A comprehensive data analysis project examining patterns and trends in data breach incidents across different industries and time periods. The analysis provides insights into attack vectors, victim demographics, and emerging threats in cybersecurity.

## Dataset

### Data Sources
- **Verizon DBIR** (Data Breach Investigations Report)
- **IBM Cost of Data Breach** reports
- **Have I Been Pwned** breach database
- **Privacy Rights Clearinghouse** breach database
- **Custom web scraping** from security news sources

### Data Size
- **500,000+** breach records
- **15+ years** of historical data
- **50+** data fields per record
- **100+** industry categories

## Data Visualization Examples

### Trend Analysis
```python
import matplotlib.pyplot as plt
import seaborn as sns

# Breach trends over time
plt.figure(figsize=(12, 6))
sns.lineplot(data=df, x='year', y='breach_count')
plt.title('Data Breach Trends by Year')
plt.xlabel('Year')
plt.ylabel('Number of Breaches')
plt.show()
```

### Industry Heatmap
```python
# Industry vs Attack Vector heatmap
pivot_table = df.pivot_table(
    values='records_lost', 
    index='industry', 
    columns='attack_vector', 
    aggfunc='count'
)
sns.heatmap(pivot_table, annot=True, fmt='d')
plt.title('Breach Frequency by Industry and Attack Vector')
plt.show()
```
